## Important ðŸ”‘words and their Use-cases:

- **CREATE:** 
    It is used to create new DATABASE, TABLE, ROW and COLUMN etc.
- **USE:** 
    It is used to use specific DATABASE.
- **INT:**
    It is used when we want to store integer(...,-1,0,1,...) type of data.
- **PRIMARY KEY:**
    A Primary Key is a field that uniquely identifies each record in a table and cannot contain NULL or duplicate values.
- **VARCHAR:** 
    VARCHAR is a data type that stores characters/strings, where the storage size depends on the actual length of the data, not the maximum length.
- **UNIQUE:** 
    Ensures all values are different, used for emails, usernames, phone numbers
- **INSERT INTO:** 
    To insert data into a table.
- **SELECT:** 
    Retrieves data from a table
- **FROM:** 
    Specifies the table name, Tells MySQL where data comes from.
- **DISTINCT:** 
    Used when you want unique results only.
- **WHERE:** 
    Used to get specific data. Filters rows based on condition
- **BETWEEN:** 
    Used for ranges (numbers, dates).
- **AND:** 
    Used to combine conditions.
- **IN:** 
    Matches values from a list

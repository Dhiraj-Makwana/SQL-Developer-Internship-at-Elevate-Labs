## Important Keywords and their usecases:

1. CREATE: It is used to create new DATABASE, TABLE, ROW and COLUMN etc.
2. USE: It is used to use specific DATABASE.
3. INT: It is used when we want to store integer(...,-1,0,1,...) type of data.
4. PRIMARY KEY: A Primary Key is a field that uniquely identifies each record in a table and cannot contain NULL or duplicate values.
5. VARCHAR: VARCHAR is a data type that stores characters/strings, where the storage size depends on the actual length of the data, not the maximum length.
6. UNIQUE: Ensures all values are different, used for emails, usernames, phone numbers
7. INSERT INTO: To insert data into a table.
8. SELECT: Retrieves data from a table
9. FROM: Specifies the table name, Tells MySQL where data comes from.
10. DISTINCT: Used when you want unique results only.
11. WHERE: Used to get specific data. Filters rows based on condition
12. BETWEEN: Used for ranges (numbers, dates).
13. AND: Used to combine conditions.
14. IN: Matches values from a list
